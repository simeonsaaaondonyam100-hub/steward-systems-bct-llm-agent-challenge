# Yelp Open Dataset Processed Subset

This folder is reserved for optional output from:

```bash
python scripts/ingest_yelp_subset.py
```

The full Yelp Open Dataset is not included in this repository. Generated `users.json`, `items.json`, and `reviews.json` files are ignored by git. The default app continues to use `data/sample/` unless `STEWARD_DATA_DIR` is pointed here.
